---

### [Euro STOXX 50 IV/RV Research](https://github.com/sil-kong/eurostoxx-iv-rv-backtest)

A reproducible empirical volatility research pipeline studying whether the implied-realized volatility spread contains information about subsequent realized variance.

**Main features:**

- realized and forward-realized volatility estimation;
- no-look-ahead IV/RV signal construction;
- stylized variance-payoff diagnostics;
- long-volatility, short-volatility and flat benchmarks;
- non-overlapping trade simulation;
- walk-forward validation;
- robustness, regime and cost-sensitivity analysis;
- automated research reports and figures.

The project explicitly tests whether apparent signal performance is explained by a naive short-volatility exposure rather than presenting backtest results as tradable alpha.

**Technologies:** Python, pandas, NumPy, matplotlib, pytest, ruff.
